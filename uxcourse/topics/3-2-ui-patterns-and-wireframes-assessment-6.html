Assessment 6

Figma Wireframes

Lesson 4 of 4

Overview

You create a set of digital wireframes in Figma as if you were preparing for a design critique with a product manager and engineering lead. You focus on applying UI patterns, structuring clear hierarchy, and showing how each screen supports the onboarding flow. You work at low fidelity, using grayscale and essential UI elements from the Figma Wireframe Kit.

You are creating the "Mid-Fidelity" version of your NSURE Onboarding flow.

I

Part 1: Setup your Figma File

  1. Create a free Figma account (if you haven't already) and open a New Design File.
  2. Name the file "NSURE Wireframes - [Your Name]".
  3. Press F (Frame tool) and select iPhone 14 (or similar) from the right-hand menu.
  4. Create 3 separate frames side-by-side. Name them: Screen 1: Start, Screen 2: Details, Screen 3: Success.

I

Part 2: Build the Structure (The "Gray Box" Phase)

Header: Use a rectangle (Gray) at the top of the frame for a navigation bar. Add a text layer "Back" on the left.

Content: Use the Text tool (T) to add a clear Headline (e.g., "Create Account").

Input Fields: Draw rectangles with a white fill and a gray stroke (border) to represent form fields. Add label text inside (e.g., "Email Address").

Primary Action: Create a button at the bottom using a Rectangle (Dark Gray) and centered white Text (e.g., "Next Step").

Tip: Select your button shape and text, then press Cmd+G (Mac) or Ctrl+G (Windows) to Group them.

I

Part 3: Prototype the Flow

Now, make it interactive.

  1. Click the Prototype tab in the top right of the right-hand panel.
  2. Select the "Next Step" button group on Screen 1.
  3. You will see a small blue circle appear on the button. Click and drag a "noodle" (connection line) from that circle to Screen 2.
  4. Repeat for Screen 2 to Screen 3.

    I

Part 4: Test and Share

  1. Click the Play Icon (Triangle) in the top right corner. This opens Presentation View.
  2. Click your buttons to verify the flow works.
  3. Click the blue Share button, select "Copy Link," and submit this URL for your assessment.

I

This activity forms the second part of the Session 3 assessment.
